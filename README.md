# Iris Flower Classification (Alfido Tech Internship)

An end-to-end Machine Learning pipeline built during my data science internship at Alfido Tech. This project processes the classic Iris dataset, scales features uniformly, handles hyperparameter constraints, evaluates multiple classification models, and serializes the top-performing production model for automated deployment.

---

## Project Overview & Pipeline Structure

The project utilizes a structured, linear pipeline contained within a single executable Jupyter Notebook (`iris_classification.ipynb`).
